#  VocalVision AI

### Read • Translate • Speak from Images

---

##  Overview

VocalVision AI is an intelligent application that extracts text from images using OCR, translates it into multiple languages, and reads it aloud using text-to-speech.

This project combines **Computer Vision, NLP, and Audio Processing** into a single interactive web application built with Streamlit.

---

## 🚀 Features

*  Extract text from images using EasyOCR
*  Translate text into multiple languages
*  Convert text to speech (audio playback in browser)
*  Display original and translated text side-by-side
*  Clean, responsive, and user-friendly UI

---

##  Tech Stack

* **Python**
* **Streamlit** – Web UI
* **EasyOCR** – Text detection
* **OpenCV** – Image preprocessing
* **Deep Translator** – Language translation
* **gTTS** – Text-to-speech

## ▶️ How to Run

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/VocalVision-AI.git

# Navigate to project folder
cd VocalVision-AI

# Install dependencies
pip install -r requirements.txt

# Run the application
streamlit run mainapp.py
```
